# The rank-64 identification

> **Theorem 7.2.**  At depth 2, $d_K = 8$, so $\dim(A) = 64 = d_K^2$. The full matrix algebra $M_8(\mathbb{R})$ is the ambient space. $\dim(\mathrm{im}) = 32$ and $\mathrm{rank}(\ker^2 \to \mathrm{im}) = 32$ (complete generation). At depth 3, $d_K = 16$, $\dim(\mathrm{im}) = 128$, but $\mathrm{rank}(\ker^2 \to \mathrm{im}) = 64 = \dim(M_8(\mathbb{R}))$. The generative rank freezes at the dimension of the depth-2 algebra. At depth 4, $\dim(\mathrm{im}) = 512$ while the rank remains 64. The freeze is permanent: the $\ker$ self-products span a subspace whose dimension is set by the spacetime algebra ($M_8$, where $\mathrm{Cl}(3,1)$ and $\mathfrak{so}(3,1)$ live) and cannot grow further.
