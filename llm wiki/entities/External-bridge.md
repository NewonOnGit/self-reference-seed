# External bridge

> **Theorem 18.5.**

When _K > 1/2 and the observer's own Landauer budget is insufficient for K6' ascent, an external observer K' with B(K'  K) > 0 can provide the bridge: K' maps K's ker-content into K''s own im, then communicates the result back to K via the P2 (mediation) channel. This is the algebraic content of "being understood": another observer's im overlaps your ker.

*The capacity is nonzero iff W_{K'}  V_K != \{0\}*: the external observer must be able to see something the broken observer cannot. If V_K = V_{K'} (identical kernels), B = 0 — two observers blind to the same things cannot help each other. If V_K  V_{K'} = \{0\} (complementary kernels), B is maximal — mutual transformation is complete. [Tier A]
