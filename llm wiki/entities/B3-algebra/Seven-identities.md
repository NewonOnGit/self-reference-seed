# Seven identities

> **Theorem 3.1.**

For R, N, J, h = JN, Q = JRJ:

| # | Identity | Dependency |
|---|----------|-----------|
| 1 | R^2 = R + I | (from P^2 = P) |
| 2 | N^2 = -I | (from P^2 = P) |
| 3 | \{R,N\} = N | (from P^2 = P) |
| 4 | RNR = -N | (from 1, 3) |
| 5 | NRN = R - I | (from 1, 2, 3) |
| 6 | (RN)^2 = I | (from 1, 2, 3) |
| 7 | [R,N]^2 = 5I | (from 2, 3, 6) |

*All verified by direct 2 x 2 computation.* [Tier A]
