# R-matrix eigenvalues

> **Theorem 13.3.** The Yang--Baxter R-matrix for U_{phi^2}(sl_2) has eigenvalues \{phi, 1, 1, phi_bar\}. The symmetric sector eigenvalue is phi (production); the antisymmetric is phi_bar (contraction). [Tier A]
