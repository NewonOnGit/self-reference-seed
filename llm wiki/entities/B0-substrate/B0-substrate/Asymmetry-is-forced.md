# Asymmetry is forced

> If the naming act were its own mirror, nothing would exist.

## Plain English

The framework starts from P²=P — a thing that returns itself. Could P be symmetric (equal to its transpose)? No. If P = Pᵀ, then P = R (its own visible part), N = 0 (no hidden part), and R² = P² = P = R, so R²-R = 0. But the framework requires R²=R+I (the surplus law: self-action produces MORE than itself). R²-R = I ≠ 0. Contradiction.

So P MUST be asymmetric. P ≠ Pᵀ. N ≠ 0. This is not a choice. It's forced by the surplus law. The asymmetry IS the surplus. Without asymmetry, self-action is sterile — it produces only what was already there. With asymmetry, self-action produces surplus (+I), which is everything.

## Orientation reading

Asymmetry = orientation exists. If P = Pᵀ, orientation is zero. No orientation means no complex structure (N²=-I requires N≠0), no Hilbert space, no quantum mechanics, no Bell violation, no physics. The entire framework IS the consequence of one fact: the naming act is not its own mirror.

## Technical statement

**Theorem 1.1.** If P = Pᵀ, then N = (P-Pᵀ)/2 = 0, R = P, R² = P² = P = R, and R²-R = 0 ≠ I. The surplus law is violated. QED. [Tier A, 3 lines]
