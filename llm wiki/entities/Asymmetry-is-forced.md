# Asymmetry is forced

> **Theorem 1.1.**

If P = P^T, then N = (P-P^T)/2 = 0, R = P, R^2 = P^2 = P = R, so R^2 - R = 0 != I. The surplus law R^2 = R+I requires P != P^T. Asymmetry IS the surplus. QED
