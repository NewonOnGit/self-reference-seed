# Kernel concentration

> **Theorem 18.1.**

For an observer K with ker(L_K) = V_K  A, define the concentration index:

_K = (\|P_K rho P_K\|)/(\|P_K\| * \|rho\|)

where P_K is the projection onto V_K and rho is the observer's state density (the current distribution of activity across A). When _K = ker/A = 1/2 (uniform), the state is balanced. When _K > 1/2, activity is concentrated in the kernel — the observer is "trapped in its own blind spot."
