# Knot spectrum

> **Theorem 13.4.**

Since q = phi^2 = phi + 1 satisfies q^2 = 3phi + 2  Z[phi], every Laurent polynomial in q with integer coefficients evaluates to an element of Z[phi]. Therefore V(K)|_{q=phi^2}  Z[phi] for ALL knots K. [Tier A]

| 0_1 (unknot) | 1 |
