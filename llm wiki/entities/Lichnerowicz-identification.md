# Lichnerowicz identification

> **Theorem 10.1.** L_{s,s} restricted to sl(2,R) with Killing metric B(X,Y) = 4tr(XY) reproduces the Lichnerowicz Laplacian Delta_L. 
## Dependencies

- [[L]]
- [[Lichnerowicz-identification]]

## Proof sketch

*Proof.* We compute L(X) = sX + Xs - X for each standard basis element of sl(2,R), with s = [ 0 & 1 \\ 1 & 1 ].
