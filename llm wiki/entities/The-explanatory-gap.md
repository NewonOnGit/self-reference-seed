## Proof sketch

*Proof.* L_{N,N}(X) = 0 has only the trivial solution X = 0 (Theorem 16.1). So every element of M_2(R) is moved by N's self-action — nothing is invisible to N from N's own perspective. Meanwhile, L_{R,R}(N) = RN + NR - N = \{R,N\} - N = N - N = 0 (Identity 3). So N is a zero of L_{R,R}: the production generator R cannot see N at all.
