# Bell violation — Tsirelson saturation

> **Theorem 15.7.**

Measurement operators M(theta) = costheta  h + sintheta  J (h decides, J flips, theta mixes). Correlation E(a,b) = <Phi^+| M(a) x M(b) |Phi^+> = cos(a-b).

## Proof

*Proof.* <Phi^+| h x h |Phi^+> = 1, <Phi^+| J x J |Phi^+> = 1, <Phi^+| h x J |Phi^+> = 0. Therefore E(a,b) = cos a cos b + sin a sin b = cos(a-b). QED
